Good Job on the improvements done to the page/design. The html looks succinct and clean. The layout also feels more solid

Here are a couple things would like for you to work and improve.

  • Use suggestive class names. A class name is like ".a" is very generic and could be of no meaning to you after months. So if the a tag is serving as a clickable link, then you could use class name as .link instead. There are a few conventional ways to name your css classes, and we will talk into details about that later. For now try on your own to find better names of the classes, and look at other websites and see what they use.
  • To make the "flex" container more adaptive to different screen size you can further instruct on what should be done when the content extends beyond the viewport. It is called flex wrapping.
    If you want the items to always fit into its container, then you can specify no wrapping.
    so try playing with "flex-wrap:" css property to see how you can make the contents fit to any viewport.
  • Responsive web design is the type that adapts itself to the screen, by changing size, layout, etc. In order to make a responsive design, avoid using a lot of px for width and heights and font sizes. You can consider others units like %, rem, vh,vw,etc.
